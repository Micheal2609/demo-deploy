## Một số lệnh thường dùng:
```bash
scp local_file_path username@remote_host:remote_file_path
```
=> Sao chép tệp từ máy cục bộ đến máy chủ từ xa.
```bash
scp D:\MyWorkspace\my-portfolio\project\learning\demo-deploy\docker-compose.yaml maidat@192.168.100.170:/home/maidat/app
```
```bash
scp D:\MyWorkspace\my-portfolio\project\learning\demo-deploy\nginx.conf maidat@192.168.100.170:/home/maidat/app
```
